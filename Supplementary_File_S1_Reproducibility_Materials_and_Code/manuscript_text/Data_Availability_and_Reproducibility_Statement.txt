Data Availability and Reproducibility Statement

Due to restrictions associated with the original data source and data-use conditions, the complete infrared image dataset cannot be publicly released. To improve reproducibility, the class definitions, fixed training/validation split, annotation format examples, training configuration files, model configuration, evaluation protocol, main experimental logs, and the main reproduction script are provided as supplementary materials. The trained model weights and additional verification files can be made available from the corresponding author upon reasonable request for academic verification purposes, subject to the applicable data-use restrictions. All experiments were conducted using the fixed data split and training settings reported in this manuscript.
