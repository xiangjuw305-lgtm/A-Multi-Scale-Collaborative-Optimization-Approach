Response to Reviewer 2, Comment 1: Reproducibility

Thank you for this valuable suggestion. We agree that reproducibility is important for evaluating the reliability of the reported results. Due to restrictions associated with the original data source and data-use conditions, the complete infrared image dataset cannot be publicly released. However, to improve reproducibility, we have added a reproducibility statement in the revised manuscript and provided Supplementary File S1, which includes the class definitions, fixed training/validation split, annotation format examples, training configuration, model configuration, evaluation protocol, main experimental logs, and the main reproduction script. The script provides the main training entry for the proposed YOLOv10m-based method with Target Aug, Class Weight, SGRB-DE, C2f-SimAM, and MPDIoU. In addition, the trained model weights and additional verification files can be made available from the corresponding author upon reasonable request for academic verification purposes, subject to applicable data-use restrictions. These revisions have been added to Section 3.2 and the Data Availability and Reproducibility Statement.
